# one-bash-it

A [one.bash][] repo which wraps [bash-it][].

## Usage

1. Install [one.bash][].
2. `one repo add https://github.com/one-bash/one-bash-it`
3. Enable bash-it aliases/completions/plugins with `one` commands. For example, enable aliases.completion by `one completion enable aliases.completion`.

## Update

`one repo update one-bash-it`

## Notice

1. Do not use [bash-it commands](https://bash-it.readthedocs.io/en/latest/commands/). Use one.bash commands.
